# CUPED and VWE Experiments

Воспроизводимые эксперименты по снижению дисперсии оценки эффекта в A/B-сценариях. Репозиторий сравнивает `plain diff`, `CUPED`, `VWE` и их комбинацию на синтетических данных.

## Задача
Нужно сравнить несколько способов снижения дисперсии оценки эффекта и измерить, как они влияют на устойчивость оценок.

## Архитектура решения
```text
Генерация синтетических данных
            │
            ▼
    Расчёт эффекта четырьмя способами
 ┌──────────┼──────────┬──────────┐
 ▼          ▼          ▼          ▼
plain diff  CUPED      VWE    CUPED+VWE
 └──────────┴──────────┴──────────┘
            │
            ▼
   Оценка std(estimate) и rmse(estimate)
            │
            ▼
        Графики и таблицы
```

## Структура репозитория
- сценарии генерации данных;
- сценарии построения графиков;
- материалы с итоговыми таблицами.

## Метрики
- `std(estimate)`
- `rmse(estimate)`
- относительное снижение дисперсии по сравнению с `plain diff`

## Результаты
В текущем публичном представлении численные значения не видны. Cursor должен получить реальные числа из сценариев генерации фигур и сохранить их в:

В README нужно указать:
