# CUPED vs VWE Experiments

## Кратко
Воспроизводимые симуляции для сравнения plain diff, CUPED, VWE и их комбинации на синтетических A/B-данных. Репозиторий помогает понять, в каких условиях методы снижения дисперсии действительно улучшают оценку эффекта.

## Задача
В A/B-анализе важно не только оценить эффект, но и сделать это устойчиво к шуму. В проекте сравниваются четыре режима:
- plain diff как базовый ориентир;
- CUPED при наличии полезного предэкспериментального сигнала;
- VWE при неоднородной дисперсии;
- комбинация CUPED и VWE в смешанных сценариях.

## Что улучшено
- сравнение вынесено в единый воспроизводимый контур;
- показано, что разные методы дают выигрыш в разных режимах данных;
- добавлена возможность сравнивать устойчивость оценки, а не только среднее значение эффекта.

## Архитектура расчёта
```text
[генерация синтетических данных]
              |
              v
[plain diff | CUPED | VWE | CUPED+VWE]
              |
              v
[оценка эффекта и разброса оценки]
              |
              v
[сравнение методов по устойчивости]
```

## Критерии качества
- стандартное отклонение оценки эффекта;
- RMSE оценки эффекта;
- относительное снижение дисперсии относительно plain diff.

## Ключевой вывод
CUPED полезен, когда предэкспериментальный сигнал хорошо связан с целевой метрикой. VWE помогает, когда шум в данных неоднороден. Их сочетание может давать более устойчивую оценку эффекта в смешанных сценариях, но выигрыш зависит от структуры данных и не является универсальным.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python scripts/make_figures.py
```

## Ограничения
- результаты зависят от параметров симуляции;
- синтетические данные не заменяют проверку на реальном экспериментальном контуре;
- при переносе выводов на продовые A/B-эксперименты нужен отдельный анализ предпосылок методов.
