# CUPED and VWE Experiments

Воспроизводимые эксперименты по снижению дисперсии оценки эффекта в A/B-сценариях. Репозиторий сравнивает `plain diff`, `CUPED`, `VWE` и их комбинацию на синтетических данных.

## Назначение

Сравнить несколько способов снижения дисперсии оценки эффекта и измерить, как они влияют на устойчивость итоговой оценки.

## Архитектура

```text
Генерация синтетических данных
            │
            ▼
Расчёт эффекта четырьмя способами
 ├─ plain diff
 ├─ CUPED
 ├─ VWE
 └─ CUPED + VWE
            │
            ▼
Оценка std(estimate) и rmse(estimate)
            │
            ▼
Графики и итоговые таблицы
```

## Что реализовано

- генерация синтетических данных;
- расчёт оценки эффекта несколькими способами;
- сравнение дисперсии и ошибки оценки;
- построение графиков и итоговых таблиц.

## Структура репозитория

- `src/` — основная логика моделирования;
- `scripts/` — сценарии запуска;
- `figures/` — результаты построения графиков;
- `article_draft.md` — пояснительный текст;
- `requirements.txt`, `pyproject.toml` — зависимости.

## Оценка качества

Основные метрики:

- `std(estimate)`
- `rmse(estimate)`
- относительное снижение дисперсии по сравнению с `plain diff`

## Запуск

```bash
python -m venv .venv

pip install -r requirements.txt

```

## Ограничения

- выводы зависят от параметров генерации синтетических данных;
- для корректного сравнения фиксируют единый сценарий генерации и сохраняют итоговые таблицы рядом с графиками.
